## 文档结构说明 / Document Structure

### 索引文档（顶层导航）
这些文档用于快速定位和理解项目，特别适合AI Agent使用：
- **ARCHITECTURE.md** - 模块索引、数据流、依赖关系
- **PROJECT_MAP.md** - 文件映射、快速查找表
- **API-REFERENCE.md** - API签名、接口定义
- **FEATURES.md** - 功能清单、详细功能列表
- **CHANGELOG.md** - 开发历史、版本变更

### 用户文档（使用指南）
面向实际使用项目的用户：
- **入门系列** - getting-started.md, quick-reference.md, syntax-reference.md
- **进阶内容** - advanced-features.md

### 状态文档（项目概览）
项目当前状态和统计：
- **evolution-status.md** - 版本统计、里程碑、技术债务

## 文档贡献 / Contributing to Documentation

如果你发现文档有错误或需要改进，欢迎提交 Issue 或 Pull Request！

**文档更新指南**：
- 添加新功能时，更新 `FEATURES.md` 和 `CHANGELOG.md`
- 修改API时，更新 `API-REFERENCE.md`
- 改变架构时，更新 `ARCHITECTURE.md`
- 详细说明请参考 `.cursorrules` 中的"修改建议"部分
